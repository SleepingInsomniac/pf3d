# PD3d

[PixelFaucet](https://github.com/sleepinginsomniac/pixelfaucet) 3D Graphics library

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     pf3d:
       github: sleepinginsomniac/pf3d
   ```

2. Run `shards install`

## Usage

```crystal
require "pf3d"
```
